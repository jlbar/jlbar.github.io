Recommendations with Pyspark

15/Jun 2017

I will detail some of the basic components involved in generating recommendations using pyspark’s alternating least squares matrix factorization methods.

Installation

I used homebrew on OSX as follows.

$ brew install apache-spark

I may well have done some additional configuration work after the install, but I have since forgotten about any of that.

Shell Usage

Pyspark comes packed with a shell utility, which is callable from the command line via the pyspark command. Alternatively, it is possible to launch the pyspark shell within ipython. To use ipython, set the PYSPARK_DRIVER_IPYTHONvariable to ipython when invoking pyspark as follows.

$ PYSPARK_DRIVER_IPYTHON=ipython pyspark

Training Data

In order to train a model, we must first load data. Pyspark’s ALS.trainImplicit method expects either an RDD or tuple that admits the structure: (userID, productID, rating).
